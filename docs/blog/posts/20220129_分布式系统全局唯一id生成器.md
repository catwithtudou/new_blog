# 分布式系统全局唯一ID生成器

> 代码实现示例[repo地址](https://github.com/catwithtudou/distribute_structure_examples/tree/master/id)

使用场景一般：

- 分布式系统设计时，数据分片场景下，通常需要一个全局唯一id；

- 在消息系统中需要消息唯一ID标识来防止消息重复；

- 多系统打通需要一个全局唯一标识 （如集团各业务线面对不同用户，需要一个全局用户id）；
